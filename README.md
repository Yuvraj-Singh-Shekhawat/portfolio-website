# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my projects, technical skills, education, and experience. The website features a clean user interface, smooth animations, project showcase, and a functional contact form powered by EmailJS.

## 🔗 Live Demo
https://yuvrajsinghshekhawat.netlify.app/

## ✨ Features

- Responsive Design
- Modern Dark Theme UI
- About Me Section
- Skills & Technologies
- Education Timeline
- Project Showcase Slider
- Functional Contact Form (EmailJS)
- Resume Download
- Smooth Navigation
- Typing Animation
- Social Media Links

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- EmailJS
- Typed.js

## 🎯 Learning Outcomes

- Responsive Web Design
- JavaScript DOM Manipulation
- Project Slider Implementation
- EmailJS Integration
- Frontend Development Best Practices
- Portfolio Design & Deployment
